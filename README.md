# CPG Margin Diagnostic

**Live tool:** [cpg.jakeherridge.com](https://cpg.jakeherridge.com)

A professional-grade CPG margin analysis tool built for retail buyers, brand managers, and operations leaders. Enter your product cost structure, pricing, and channel mix to instantly surface margin health, identify risk zones, and generate a supplier negotiation brief.

## Built With
- Vanilla HTML/CSS/JS (single file, zero dependencies)
- Deployed via Vercel
